Scan Upload par JM Créa, scan your wp-upload folder and automatically detect suspicious files.

== Description ==

Scan Upload par JM Créa, scan votre dossier wp-upload et détecte automatiquement les fichiers suspects. 

Scan Upload pour Wordpress détecte les fichiers de type : 

- .PHP
- .JS
- .HTACCESS
- .SQL
- .EXE
- .ZIP
- .RAR
- .CZIP
- .TAR.GZ
- .CACHE
- .OLD
- .BACK
- .BASH

Une fois les fichiers trouvés, il est possible de les supprimer en 1 clic.

== Installation ==

1. Uploadez `Scann_Upload_Par_JM_Crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Scan Upload par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite sur ‘Outils’ dans votre menu à gauche puis cliquer sur ‘Scann Upload’
